# Frontend Workspace

Este diretorio contem o frontend React + Vite do projeto.

## Projeto ativo

Use sempre:

`Front end/vite-project`

## Comandos

```bash
cd "Front end/vite-project"
npm install
npm run dev
```

## Estrutura valida

- `src/components`
- `src/components/ui`
- `src/data`
- `src/App.jsx`
- `src/main.jsx`

Nao use arquivos fora de `src` para componentes/dados do app.
